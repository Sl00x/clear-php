Règles Description Code

Création d'un template

Pour créer une page de template il vous faut accéder au dossier (/views/) et y créer un fichier php.
Une fois ceci effectuer vous aurais juste a coder votre page en html.

Recupération de variable dans les templates

Pour récupérer les variables.
Il y a deux fonction principal pour cela:

Fonction Retoure Utilisation
get(variable) return any; Utilisée pour retourner des liste des variables et les utilisers pour comparaison ou autre.
show(variable) return string; Utilisée pour affichée une variable en text. Pour eviter d'utiliser les echo get(variable)
